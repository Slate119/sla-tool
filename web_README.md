# JSON格式化工具 - Web版本

这是一个功能完备的Web版JSON格式化工具，提供现代化的用户界面和丰富的交互功能。

## 🚀 快速开始

1. **启动Web服务器**
   ```bash
   python -m http.server 8000
   ```

2. **打开浏览器**
   访问：http://localhost:8000

## ✨ 功能特性

### 核心功能
- **JSON格式化**：将压缩的JSON转换为易读格式，支持4空格缩进
- **JSON压缩**：移除所有空格和换行符，生成紧凑格式
- **语法高亮**：自动为JSON元素添加颜色标识（键名、字符串、数字、布尔值）
- **实时语法检查**：输入时即时验证JSON格式有效性

### 操作功能
- **一键复制**：快速复制输入或输出内容到剪贴板
- **粘贴功能**：从剪贴板快速粘贴JSON数据
- **文件下载**：将格式化结果下载为.json文件
- **清空功能**：快速清空输入输出区域
- **示例加载**：提供JSON示例数据供测试使用

### 用户体验
- **字符计数**：实时显示输入输出字符数
- **状态提示**：清晰显示当前操作状态和结果
- **通知系统**：操作成功/失败的即时反馈
- **快捷键**：支持Ctrl+Enter快速处理
- **响应式设计**：适配桌面和移动端

## 📝 使用说明

1. **输入JSON数据**
   - 直接输入JSON字符串
   - 点击"粘贴"从剪贴板获取
   - 点击"示例"加载测试数据

2. **选择操作模式**
   - "格式化"：美化JSON结构
   - "压缩"：生成紧凑格式

3. **处理数据**
   - 点击"处理"按钮
   - 或使用Ctrl+Enter快捷键

4. **获取结果**
   - 查看右侧输出区域
   - 点击"复制"获取结果
   - 点击"下载"保存为文件
